# 🛒 API E-commerce

Projeto de uma API RESTful para um sistema de e-commerce, com foco em boas práticas de modelagem, organização e arquitetura de software.

---

## 📌 Status do Projeto

🚧 Em desenvolvimento — fase inicial (modelagem UML)

---

## 🎯 Objetivo

O objetivo deste projeto é:

* Construir uma API de e-commerce completa
* Aplicar boas práticas de desenvolvimento backend
* Utilizar arquitetura em camadas
* Criar um projeto sólido para portfólio

---

## 🧱 Modelagem do Sistema

O sistema foi modelado utilizando UML (Diagrama de Classes), contendo as principais entidades de um e-commerce:

* Usuário
* Produto
* Carrinho
* ItemCarrinho
* Pedido
* ItemPedido
* Pagamento

---

## 🔗 Relacionamentos principais

* Um usuário possui um carrinho
* Um carrinho possui vários itens
* Cada item do carrinho está associado a um produto
* Um pedido possui vários itens
* Cada item do pedido referencia um produto
* Um pedido possui um pagamento
